# **App Name**: ProofChain

## Core Features:

- Global Entry Portal: A welcoming landing page with clear value propositions and CTAs guiding users (Students, Institutions, Verifiers) to appropriate actions like issuing or verifying credentials.
- Secure Web3 Wallet Connect: Enables users to connect their Web3 wallet (e.g., MetaMask-style UI) for secure authentication, managing their decentralized identity, and role selection.
- University Credential Issuer: A focused dashboard providing institutions with a streamlined form UX to input student data and issue tamper-proof academic credentials to the blockchain.
- Student Credential Dashboard: A personal dashboard for students to securely view, manage, and explore detailed information for their owned academic credentials, each with an authenticity status badge.
- Instant Credential Verifier: A clean, secure page for employers and other verifiers to instantly authenticate credentials by providing a hash, uploading a file, or scanning a QR code, displaying a clear verification result.
- ProofChain Component Library: A foundational set of production-ready, reusable UI components (buttons, inputs, cards, modals, badges, etc.) developed according to a consistent design system to ensure coherence and quality.

## Style Guidelines:

- Color scheme: Dark theme, embodying credibility and precision. Background: A very dark, slightly bluish-grey neutral (#141A1F). Secondary panels/elevated surfaces are a slightly lighter variant (#1E252C).
- Primary action color: A vibrant yet clear blue (#3DA3FF), inspired by an indigo/cyan hybrid, used sparingly for buttons, highlights, and focus states. This color is balanced to pop against the dark background without being over-saturated.
- Accent color: A muted deep blue-violet (#636399), analogous to the primary color, used for secondary emphasis and complementary elements, ensuring visual harmony and sophisticated contrast.
- Text hierarchy: Primary text is a light, near-white grey (#E6EBEE); secondary text for muted information (#96A0A5); and tertiary text for low emphasis (#565E62). Status colors include Success (#39A339), Warning (#E8B83A), and Error (#E64A4A).
- Font: 'Inter' (sans-serif) for all text elements. It provides a clean, modern, and highly readable foundation. Typography will feature a strong hierarchy: Hero sections use a bold, large weight; Section titles a medium weight; Body text is neutral and optimized for readability. Note: currently only Google Fonts are supported.
- Minimalistic, stroke-based iconography will be used carefully to reinforce meaning without adding unnecessary visual clutter. Icons should be clear, concise, and professional, avoiding any playful or unserious styles to maintain trust and credibility.
- The layout prioritizes clarity and intentionality, using a consistent spacing scale (4px, 8px, 12px, 16px, 24px, 32px). Depth is conveyed through subtle layering and soft, low-contrast borders rather than heavy shadows. Components feature subtle rounded corners to maintain a polished yet precise feel.
- Micro-interactions are fast and smooth (150-250ms), such as a subtle scale and color shift on button hover, a soft lift effect on card hover, and engaging loading states implemented with skeletons or spinners to ensure a responsive and intuitive user experience.
